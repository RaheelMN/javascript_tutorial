Primitive data types

Integer, string, boolean, null and undefined are primitive datatype stored in stack

Except null and undefined primitives are assigned object

Primitive datatypes are immutable. That means if we change the value of variable

then it will store it in new address and variable will point to that address and last address

will be deleted

Object datatype

Objects, functions, array, regex are object datatype

They store address of heap memory where object is stored
